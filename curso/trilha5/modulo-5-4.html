Projeto 4: Automatizando Workspace com gws

Gmail, Drive, Sheets e Calendar operados diretamente pelo terminal. Conecte o Google Workspace inteiro a agentes CLI para automacao real de trabalho.

45 Minutos
Intermediario Nivel
Hands-on Tipo
1

🔑 Setup do gws

O gws (Google Workspace CLI) precisa de um Google Cloud Project com OAuth configurado. Vamos configurar tudo passo a passo.

Instalacao e autenticacao

1. Instale o gws globalmente:

npm install -g @anthropic-ai/gws

2. Configure o Google Cloud Project:

# Acesse console.cloud.google.com
# 1. Crie um novo projeto (ou use existente)
# 2. Ative as APIs: Gmail, Drive, Sheets, Calendar
# 3. Configure OAuth consent screen (tipo: External)
# 4. Crie credenciais OAuth 2.0 (tipo: Desktop app)
# 5. Baixe o client_secret.json

3. Autentique:

gws auth login --client-secret ./client_secret.json
# Abre o navegador para autorizacao OAuth

4. Verifique:

gws auth status
# ✓ Authenticated as: seu-email@gmail.com
# ✓ Scopes: gmail, drive, sheets, calendar
2

📧 Gmail via CLI

Leia, envie e filtre emails diretamente pelo terminal. Email vira dado estruturado que agentes conseguem processar.

Operacoes principais

Listar emails recentes:

gws gmail messages list --maxResults 5 --json

Ler um email especifico:

gws gmail messages get MSG_ID --json

Enviar email:

gws gmail messages send \
  --to "colega@empresa.com" \
  --subject "Relatorio semanal" \
  --body "Segue o relatorio em anexo."

Filtrar por label e data:

gws gmail messages list \
  --query "label:INBOX is:unread after:2026/03/20" \
  --json

📊 Por que JSON importa

A flag --json retorna dados estruturados que agentes conseguem processar programaticamente. Sem ela, o output e texto formatado para humanos — bonito de ler, mas dificil de parsear. Agentes precisam de JSON.

3

📁 Drive via CLI

Upload, download, busca e compartilhamento de arquivos — Google Drive operavel pelo terminal.

Operacoes de arquivo

Listar arquivos:

gws drive files list --json

Upload de arquivo:

gws drive files create \
  --name "relatorio-marco.pdf" \
  --file ./relatorio-marco.pdf

Download:

gws drive files get FILE_ID --download

Busca por nome:

gws drive files list \
  --q "name contains 'relatorio'" --json

Compartilhamento

# Compartilhar com permissao de leitura
gws drive permissions create FILE_ID \
  --role reader \
  --type user \
  --emailAddress "colega@empresa.com"

# Gerar link publico
gws drive permissions create FILE_ID \
  --role reader \
  --type anyone

📊 Sheets via CLI

Google Sheets como banco de dados leve via CLI — leia, escreva e manipule planilhas diretamente do terminal.

Leitura e escrita

Ler dados de uma planilha:

gws sheets values get SHEET_ID \
  --range "A1:D10" --json

Escrever dados:

gws sheets values update SHEET_ID \
  --range "A1" \
  --values '[["Nome","Email","Status"],["Ana","ana@ex.com","Ativo"]]'

Adicionar linhas (append):

gws sheets values append SHEET_ID \
  --range "A:D" \
  --values '[["Pedro","pedro@ex.com","Pendente"]]'

💡 Sheets como banco de dados

Para projetos pequenos, Google Sheets funciona surpreendentemente bem como banco de dados: e gratuito, tem UI amigavel para nao-devs, e acessivel via API. Via CLI, um agente pode ler planilhas de clientes, atualizar status de pedidos e gerar relatorios — tudo sem banco de dados tradicional.

📅 Calendar via CLI

Crie, liste e gerencie eventos do Google Calendar. Agenda operavel por agente — marque reunioes, verifique disponibilidade, envie convites.

Operacoes de calendario

Listar eventos da semana:

gws calendar events list \
  --timeMin "2026-03-23T00:00:00Z" \
  --timeMax "2026-03-29T23:59:59Z" \
  --json

Criar evento:

gws calendar events insert \
  --summary "Reuniao de Sprint" \
  --start "2026-03-25T10:00:00" \
  --end "2026-03-25T11:00:00" \
  --attendees "time@empresa.com"

Deletar evento:

gws calendar events delete EVENT_ID

Exemplo de output JSON

{
  "id": "abc123",
  "summary": "Reuniao de Sprint",
  "start": { "dateTime": "2026-03-25T10:00:00-03:00" },
  "end": { "dateTime": "2026-03-25T11:00:00-03:00" },
  "attendees": [
    { "email": "time@empresa.com", "responseStatus": "needsAction" }
  ]
}

🤖 gws + Gemini CLI

A combinacao mais poderosa: gws como extensao do Gemini CLI. O agente orquestra todo o Google Workspace com linguagem natural.

Configuracao como extensao

# Instale gws como extensao do Gemini CLI
gemini extensions install @anthropic-ai/gws

# Verifique
gemini extensions list
# ✓ gws - Google Workspace CLI (gmail, drive, sheets, calendar)

Workflow completo via agente

> Me mostre os emails nao lidos de hoje
> Leia o email do cliente sobre a reuniao e crie um evento no calendario para a data mencionada
> Pegue os dados da planilha "Clientes" e envie um email resumo para cada cliente ativo

💡 Automacao de conhecimento

Isso nao e apenas automacao de tarefas — e automacao de conhecimento. O agente le um email, entende o conteudo, extrai informacoes relevantes (datas, nomes, acoes) e age em multiplos servicos. Um humano levaria 15 minutos para fazer manualmente o que o agente faz em 30 segundos.

Resumo Final

✓ gws conecta Gmail, Drive, Sheets e Calendar ao terminal via OAuth
✓ Flag --json transforma dados do Workspace em formato processavel por agentes
✓ Sheets funciona como banco de dados leve para projetos pequenos e medios
✓ Calendar via CLI permite agentes criarem e gerenciarem eventos autonomamente
✓ gws + Gemini CLI cria automacao de conhecimento — o agente entende e age em multiplos servicos
